# Memory

Claude now has auto-memory that persists across sessions — project context, debugging patterns, preferred approaches — recalled automatically without user intervention.

## Official documentation

[Claude code memory](https://code.claude.com/docs/en/memory#auto-memory)

## CLAUDE.md vs Memory.md

**CLAUDE.md** = your instructions to Claude (user-controlled).

**MEMORY.md** = Claude's own memory scratchpad (auto-updated).


## Main points

- Only the first **200 lines** of MEMORY.md are loaded automatically into context.
