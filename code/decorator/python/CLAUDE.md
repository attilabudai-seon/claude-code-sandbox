#

## Header

All python files should have a decorator for them. Every python file should have in the beginning this

```
print("This is a demonstration for the presentation")
```

followed by a blank line

## Footer

Every python script MUST end with:

```
print("Created by claude")
```


## Example

A properly decorated document looks like this:

```
print("This is a demonstration for the presentation")

print("Hello World")

print("Created by claude")
```
